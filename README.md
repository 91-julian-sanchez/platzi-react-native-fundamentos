# HabitosApp

Bienvenido a **HabitosApp**, una aplicación diseñada para ayudarte a formar y mantener hábitos saludables.

## Descripción del Proyecto

HabitosApp es una aplicación construida con [React Native](https://reactnative.dev/) y [Expo](https://expo.dev). Esta aplicación te permite:
- Crear y gestionar hábitos.
- Realizar un seguimiento de tu progreso.
- Recibir recordatorios y motivación.

Este proyecto fue desarrollado como parte del curso [Fundamentos de React Native](https://platzi.com/cursos/fundamentos-react-native/) en Platzi.

## Prerequisitos

Antes de comenzar, asegúrate de tener instalado lo siguiente:
- **Node.js** (versión 14 o superior) - [Descargar](https://nodejs.org/)
- **npm** o **yarn** como gestor de paquetes
- **Expo CLI** - Se instalará al ejecutar `npm install`
- Un emulador de Android/iOS o un dispositivo móvil con la aplicación Expo Go instalada

## Comenzar

Para comenzar a trabajar con HabitosApp, sigue estos pasos:

1. Instala las dependencias

   ```bash
   npm install
   ```

2. Inicia la aplicación

   ```bash
   npx expo start
   ```

En la salida, encontrarás opciones para abrir la aplicación en un emulador o en tu dispositivo móvil.
